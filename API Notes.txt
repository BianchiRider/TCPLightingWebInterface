Some simple documentation for the API calls. These are handy to know if you plan on integrating this project with IFTTT

{host} = Your host, IE your IP address
:{port} = If you're running on port 80, you dont need this. If you're specifying a different remote port, or a non standard port, specify it here
{level} = Numeric value from 1 - 100
{amount} = Numeric value from 1 - 100


//Device ON
{host}:{port}/api.php?fx=toggle&type=device&uid={deviceID}&val=1

//Device OFF
{host}:{port}/api.php?fx=toggle&type=device&uid={deviceID}&val=0

//Dim Device To Brightness
{host}:{port}/api.php?fx=dim&type=device&uid={deviceID}&val={level}

//Increase Device Brightness
{host}:{port}/api.php?fx=brightenby&type=device&uid={deviceID}&val={amount}

//Decrease Device Brightness
{host}:{port}/api.php?fx=dimby&type=device&uid={deviceID}&val={amount}



//Room OFF
{host}:{port}/api.php?fx=toggle&type=room&uid={roomID}&val=0

//Room ON
{host}:{port}/api.php?fx=toggle&type=room&uid={roomID}&val=1

//Room Dim To
{host}:{port}/api.php?fx=dim&type=room&uid={roomID}&val={level}

//Increase Room Brightness
{host}:{port}/api.php?fx=brightenby&type=room&uid={roomID}&val={amount}

//Decrease Room Brightness
{host}:{port}/api.php?fx=dimby&type=room&uid={roomID}&val={amount}


//Home ON
{host}:{port}/api.php?fx=toggle&type=all&uid=ALL&val=1

//Home OFF
{host}:{port}/api.php?fx=toggle&type=all&uid=ALL&val=0

//Home Dim To
{host}:{port}/api.php?fx=dim&type=all&uid=all&val={level}

//Home Increase Brightness
{host}:{port}/api.php?fx=brightenby&type=all&uid=all&val={amount}

//Home Decrease Brightness
{host}:{port}/api.php?fx=dimby&type=all&uid=all&val={amount}


NOTE** If you want to force a password for the remote API Calls, at the end of each API Call append &password={yourPassword}
